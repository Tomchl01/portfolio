Live Earthquake Map

Real-time seismic activity visualization using data from the USGS Earthquake API. Focused on Costa Rica and Central America region.

About This Project

This interactive map displays recent earthquake activity, pulling live data from the United States Geological Survey (USGS). The visualization helps identify patterns in seismic activity and provides insights into earthquake magnitude, depth, and location.
